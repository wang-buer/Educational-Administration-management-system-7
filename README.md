## 1、项目介绍

教务（选课）管理系统7拥有三个角色

- 管理员：公告管理、学生管理、教师管理、查看所有课程、学生选课管理
- 教师：查看公告、查看我的课程、成绩评定、发布课程、通讯录
- 学生：选课、退课、查看课程、通讯录


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：Layui、jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- 是否为maven项目：否
- 硬件环境：Windows 或者 Mac OS
